HTML - Hyper Text Markup Langauage



HTML(Hyper Text Markup Language) is the standard markup language for webpages.It is used to display documents in a web browser. It was first developed by Tim Berners Lee in 1990 while working at the European Organization for Nuclear Research (CERN). It was one of the key technologies used to build and publish the first website on August 6, 1991. HTML describes the structure of a webpage and consists of different elements.These HTML elements tell the browser how to display the webpage. HTML is usually assisted by lanaguages like Cascading Style Sheets(CSS) and scripting lanaguages like JavaScript.

HTML Features

HTML
  • Links - embed a URL link into a web page
  • Lists - organize items into a list
  • Images - embed a photograph or drawing into a web page
  • Block-Level Elements - elements that takes up the entire width of a page
  • Inline Elemenst - formats the inner content of block-level elements such as adding links and emphasized strings.

    • Contributions


    HTML has a lot of uses and contributions and is used in some way shape or form by most people in our world today. It is used for web development. Developers use HTML code to build and design how a browser displays web page elements such as text, images, hyperlinks and other media files. It is used for navigation of the Internet. Users can easily naviagte and insert links between related pages and webstes because HTML is often used to embed hyperlinks. HTML is also used for web documentation. HTML makes it possible to organize and format documents similar to word-processing applications like Microsoft Word.

    Simplified History

  • 1989 - Tim Berners-Lee invented www
  • 1991 - HTML first developed by Tim Berners Lee
  • 1997 - HTML 3.2, first common standard for HTML
  • 1998 - HTML 4.01
    • - specified a clear relationship between HTML and JavaScript
    - introduced stylesheets
  • 2000 - XHTML 1.0, basically HTML 4.01 with strict syntax standards of XML
  • 2008 - HTML 5, combines XHTML, CSS3, MathML, SVG
  • 2016 - HTML 5.1
  • 2017- HTML 5.2
